Program Description:

  FlippingDeadmau5.java:
    This program creates a deadmau5 object on the canvas when the user clicks
  somewhere on it. This program also allows the user to move the deadmau5
  around, flip the deadmau5 on itself when clicked on the face with no movement
  for half a second, flip the deadmau5 onto its right with respects to the 
  center of the right ear  when clicked anywhere on the right ear with no 
  movement for half a second, and flip the deadmau5 onto its left with 
  respects to the center of the left ear when clicked anywhere on the left ear
  with no movement for half a second. If the user clicks in the location where
  the ear and the face overlap for half a second, deadmau5 just flips over on 
  itself. The timer used to detect whether the mouse was clicked for more than
  half a seecond is created in the beginning, reset every time the user
  clicks down, and retrieved whenever the user releases the click. If deadmau5
  is moved around during any part of this, the actions deadmau5 does described
  above will not take place. A ton of if statements are used in order to check
  where the mouse was pressed and what happens if it was pressed for half a 
  second or longer. I tested this program by running it a million times
  and I am pretty sure it is tested well enough.

  Deadmau5.java:
    This program defines what makes up a deadmau5 (the ears, mouth, face, and
  eyes), where each of these peices go, where the user clicked on the deadmau5
  object, and how the deadmaus flips (to the right, to the left, on itself).
  This program is called by FlippingDeadmau5.java in order to check where the
  mouse is pressed on the deadmau5 object. When the mouse leaves the canvas
  this program removes deadmau5 from it. When deadmau5 is dragged this program
  moves deadmau5 with respects to the mouse pointer.

  Timer.java
    This program specifies how the timer object created in FlippingDeadmau5.java
  works. It starts running right when it is created (when the program first
  starts) and is reset whever the mouse is pressed and the elapsed time in
  milliseconds is retreived when the mouse press is released.
