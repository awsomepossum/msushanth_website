Program Description:

  Triangles.java
    This program creates and displays alternating triangle patterns side by side
  by outputting " "  and "*" through the use of nested while loops. First, this
  program prompts the user to enter a number. If that number is not between the
  minimum (2) and maximum (19) values, then the program asks the user to enter
  another number and keeps asking until a valid number is entered. The user can
  exit out of the program without entering a number by pressing control c. If a
  valid number is entered then the number is used to create 4 different
  triangles of that size. It runs through nested while loops. The first while
  loop runs through each triangle separately and goes to the next row when it is
  completed. The second, nested while loop, prints out the appropriate number of
  spaces and astericks in each of the triangles. The number of first characters
  is decreased every time in the first while loop in order to create the image 
  of the triangle. This is done because each triangle has 1 less or one more
  asterisk or space than the one before it.
