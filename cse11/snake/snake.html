Program Description:

  SnakeController.java:
    This program gets the arguments passed in through the command line and
creates the game's canvas using those values. An error gets thrown if the user
does not enter the appropriate values and the program will exit and not run.
If the canvas is created, then the new game button, the score label, and the
high score label will be created. An instance of the snake head will be
created here in the beginning. This class also defines the methods to deal
with the pause (space button), placing the apple in a random location, setting
and resetting the score and high score accordingly, the new game button, and
dealing with if the user won the game or if he lost. This program deals with
placing the apple in a random location on the canvas that is not in a location
where a snake segment already is. It does this by getting all the locations on
the canvas where an apple can be placed and removing all the places where the
snake is located. This file is responsible for updating the score whenever an
apple is eaten and the highscore when the game ends and if it is higher than
the current score. The new button will reset the game to how it was like in
the beginning but it will not update the highscore if the score is greater
than it. 

  Snake.java:
    This java file creates the snake head and the snake body parts. This
program also deals with moving the snake and all of its body parts
appropriately. It makes calls to the controller through the use of a reference
which tells the controller to diplay certain text at appropriate times, when
to create an apple, and the locations of all the snake pieces. If the number
of open spaces is not 0 and the snake crashes into the wall or eats itself,
this file will send a message to the controller to display the game over text.
If the number of open spaces is zero, then this file will send a message to 
the controller to display the "You Won!" text. This file  contains the methods
that makes the snake appear like it is moving. Depending on the delay the user
sets, the snake will go at that speed. If the snake is in the same location as
an apple, then this file sends a message to the controller and tells it to
update the score by 100 points, remove the apple, and place it in a new
location where there is no snake segment.
