Program Description: 

  CS11TurtleGraphics_Threaded.java:
    This program uses simple turtle graphics to create a drawing that has my
  cse 11 username followed by "JAVA" and "UC SAN DIEGO" on three separate
  lines. For each letter there is a method that defines how the turtle will
  move around to create the letter. Also all the letters will begin drawing at
  the same time in the beginning, each with their own turtle and separate
  color from the letters around it.


  EC_TestRational.java:
    In this program I created my own test cases to test out the reciprocal,
  round up, and round down methods. I tested using a mix of negative and
  positive number, big numbers, and 0. I also tested some edge/corner cases.
  Some of the cases I tested thrown an exception which is normal because the
  denominator (or numerator if getting the reciprocal) is 0.
