Program Description:

  Shape.java:
    This class defines what makes up the abstract class Shape. It defines
  the abstract methods to be used in every class that implements Shape. It
  saves the name of all the Objects that implement Shape.


  Point.java:
    This class defines what makes up a Point and what the program should do
  with it. It can create a Point, move it, copy it, and create a string
  representation of it.


  CSE11_Line.java:
    This class defines what makes up a CSE_Line and what the program should do
  with it. It can create a CSE11_Line, move it, copy it, and create a string
  representation of it.


  Circle.java:
    This class defines what makes up a Circle and what the program should do
  with it. It can create a Circle, move it, copy it, and create a string
  representation of it.


  ARectangle.java:
    This class defines what makes up the abstract class ARectangle. It
  contains the upperLeft point for the shape in the classes that extend
  ARectangle. It can create a upperLeftCorner, move it, copy it, and create a
  string representation of it.


  Rectangle.java:
    This class defines what makes up a Rectangle and what the program should
  do with it. It can create a Rectangle, copy it, and create a string
  representation of it.


  Square.java:
    This class defines what makes up a Square and what the program should
  do with it. It can create a Square, copy it, and create a string
  representation of it.


  Triangle.java:
    This class defines what makes up a Triangle and what the program should do
  with it. It can create a Triangle, move it, copy it, and create a string
  representation of it.


  CSE11_Polygon.java:
    This class defines what makes up a CSE11_Polygon and what the program
  should do with it. It takes in an array of points and creates a shape out of
  them by connecting the dots in order. It can create a CSE11_Polygon, move
  it, copy it, and create a string representation of it.


  TestCSE11_Polygon.java:
    This class defines all the methods that are used to test the CSE11_Polygon
  program. It creates Points in the correct order to draw the shape
  appropriately. Then, it passes those points into the CSE11_Polygon
  constructor through an array so it can create the shape. It tests the copy
  constructor and displays the string representation of the CSE11_Polygon and
  all the Points of its verticies.
