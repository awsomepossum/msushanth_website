Program Description:

OddEven:
  This program prompts the user to enter a series of whatever numbers they
desire. After the user is done he can press EOF to quit (control + D). The
program then sorts through all the numbers and finds the largest odd integer and
the smallest even integer. If the user typed in only odd numbers and quit, the
program will only show the largest odd number. If the user typed in only even
numbers and quit, the program will only show te smallest even number. If the
user didn't enter any numbers at all, the program will display "no numbers
entered."
