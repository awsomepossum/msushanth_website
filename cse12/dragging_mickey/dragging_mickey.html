Program Description:

DraggingMickey:
  This program creates a black mickey silhouette centered at the mouse click on
the canvas. The user can then drag the mickey around the canvas. When the mickey
is being dragged it will turn red. When the mickey is done being dragged it will
turn back to black. When the mouse pointer leaves the canvas it clears the
mickey that was on it. When the mouse re-enters the canvas the instructions
appear again and the user can press down again in order to create another mickey.
